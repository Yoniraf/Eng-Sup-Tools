SyncApp ERP API Runner (Desktop)

Download the desktop app to run SyncApp ERP APIs without browser CORS/CORP issues.

Why a desktop app?

SyncApp ERP endpoints respond with headers like Cross-Origin-Resource-Policy: same-origin. That makes browser requests from the Team Toolbox (GitHub Pages) fail with TypeError: Failed to fetch.

The desktop app starts a local proxy automatically and opens the same runner UI, so it behaves like the old .exe: it just works.

How to use

After installing, open SyncApp ERP API Runner, paste your bearer token, choose ERP/action, and run. The app aggregates all responses into one valid JSON with a copy button (same as the web UI).
